# PROJET_AJC
Environnement et OutilsPlateforme : Mainframe (Grands Systèmes): JCL, COBOL, DB2, Structure de la Base de Données  / Le projet s'appuie sur plusieurs tables (Clients, Commandes, Produits, Employés, Départements, ) dont le modèle relationnel est fourni .Fichiers de Données/ Le projet utilise plusieurs fichiers séquentiels en entrée et en sortie
Entrée PROJET.NEWPRODS.DATA Nouveaux produits à importer (format CSV) PROJET.VENTESEU.DATAVentes réalisées par les prestataires en Europe PROJET.VENTESAS.DATA Ventes réalisées par les prestataires en Asie /PROJET.STOCKS.DATA Seuils de réapprovisionnement et quantités à commander Sortie PROJET.NEWPRODS.ERRRapport des enregistrements de produits non conformes / SortiePROJET.REAPPRO.DATAListe des produits sous le seuil à commander (réapprovisionnement) Sortie PROJET.FACTURES.DATA   /Fichier contenant l'ensemble des factures générées Sortie PROJET.STATS.DATA  Fichier d'édition contenant toutes les statistiques demandées Sortie PROJET.JOURNALISATION.DATA Fichier de journalisation des anomalies rencontrées
